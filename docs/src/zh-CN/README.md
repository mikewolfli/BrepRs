# BrepRs 文档

欢迎使用 BrepRs 文档！BrepRs 是一个现代化、高性能的边界表示（BRep）库，使用 Rust 编写，专为 CAD/CAE/CAM 应用程序设计。

## 什么是 BrepRs？

BrepRs 是一个边界表示（BRep）库，提供：

- **高性能** 几何和拓扑操作
- **多语言支持** 通过语言绑定
- **全面的 API** 用于 3D 建模
- **跨平台** 兼容性
- **WebAssembly 支持** 用于浏览器应用

## 主要特性

### 核心特性

- **布尔运算**（融合、切割、相交、截面）
- **曲面建模**（平面、圆柱、球体、圆锥、圆环）
- **拓扑操作**（顶点、边、面、壳、实体操作）
- **网格生成** 和操作
- **文件 I/O**（STEP、IGES、STL、OBJ、glTF）
- **国际化** 支持

### 语言绑定

BrepRs 为多种编程语言提供绑定：

- **Python** - 使用 PyO3
- **C/C++** - 使用 FFI
- **Fortran** - 使用 C FFI
- **Java** - 使用 JNI
- **Node.js** - 使用 NAPI-RS
- **PHP** - 使用 ext-php-rs

### 性能

- **基于 Rust** 以获得最大性能
- **并行处理** 使用 Rayon
- **内存安全** 设计
- **零拷贝** 操作（尽可能）
- **优化算法** 用于常见操作

## 使用场景

BrepRs 适用于广泛的应用场景：

- **CAD 系统** - 计算机辅助设计
- **CAM 系统** - 计算机辅助制造
- **CAE 系统** - 计算机辅助工程
- **3D 可视化** - 渲染和模拟
- **几何处理** - 分析和操作
- **增材制造** - 3D 打印准备
- **机器人** - 路径规划和碰撞检测
