# BrepRs 文檔

歡迎使用 BrepRs 文檔！BrepRs 是一個現代化、高性能的邊界表示（BRep）庫，使用 Rust 編寫，專為 CAD/CAE/CAM 應用程式設計。

## 什麼是 BrepRs？

BrepRs 是一個邊界表示（BRep）庫，提供：

- **高性能** 幾何和拓撲操作
- **多語言支援** 通過語言綁定
- **全面的 API** 用於 3D 建模
- **跨平台** 相容性
- **WebAssembly 支援** 用於瀏覽器應用

## 主要特性

### 核心特性

- **布林運算**（融合、切割、相交、截面）
- **曲面建模**（平面、圓柱、球體、圓錐、圓環）
- **拓撲操作**（頂點、邊、面、殼、實體操作）
- **網格生成** 和操作
- **檔案 I/O**（STEP、IGES、STL、OBJ、glTF）
- **國際化** 支援

### 語言綁定

BrepRs 為多種程式語言提供綁定：

- **Python** - 使用 PyO3
- **C/C++** - 使用 FFI
- **Fortran** - 使用 C FFI
- **Java** - 使用 JNI
- **Node.js** - 使用 NAPI-RS
- **PHP** - 使用 ext-php-rs

### 性能

- **基於 Rust** 以獲得最大性能
- **並行處理** 使用 Rayon
- **記憶體安全** 設計
- **零複製** 操作（盡可能）
- **優化演算法** 用於常見操作

## 使用場景

BrepRs 適用於廣泛的應用場景：

- **CAD 系統** - 電腦輔助設計
- **CAM 系統** - 電腦輔助製造
- **CAE 系統** - 電腦輔助工程
- **3D 可視化** - 渲染和模擬
- **幾何處理** - 分析和操作
- **增材製造** - 3D 列印準備
- **機器人** - 路徑規劃和碰撞檢測
