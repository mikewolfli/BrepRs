# BrepRs Documentation

Welcome to the BrepRs documentation! BrepRs is a modern, high-performance boundary representation (BRep) library written in Rust, designed for CAD/CAE/CAM applications.

## What is BrepRs?

BrepRs is a boundary representation (BRep) library that provides:

- **High-performance** geometry and topology operations
- **Multi-language support** through language bindings
- **Comprehensive API** for 3D modeling
- **Cross-platform** compatibility
- **WebAssembly support** for browser applications

## Key Features

### Core Features

- **Boolean operations** (fuse, cut, common, section)
- **Surface modeling** (planes, cylinders, spheres, cones, toruses)
- **Topology operations** (vertex, edge, face, shell, solid manipulation)
- **Mesh generation** and manipulation
- **File I/O** (STEP, IGES, STL, OBJ, glTF)
- **Internationalization** support

### Language Bindings

BrepRs provides bindings for multiple programming languages:

- **Python** - Using PyO3
- **C/C++** - Using FFI
- **Fortran** - Using C FFI
- **Java** - Using JNI
- **Node.js** - Using NAPI-RS
- **PHP** - Using ext-php-rs

### Performance

- **Rust-based** for maximum performance
- **Parallel processing** with Rayon
- **Memory-safe** design
- **Zero-copy** operations where possible
- **Optimized algorithms** for common operations

## Use Cases

BrepRs is suitable for a wide range of applications:

- **CAD systems** - Computer-aided design
- **CAM systems** - Computer-aided manufacturing
- **CAE systems** - Computer-aided engineering
- **3D visualization** - Rendering and simulation
- **Geometry processing** - Analysis and manipulation
- **Additive manufacturing** - 3D printing preparation
- **Robotics** - Path planning and collision detection
