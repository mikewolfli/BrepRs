# Документация BrepRs

Добро пожаловать в документацию BrepRs! BrepRs — это современная высокопроизводительная библиотека граничного представления (BRep), написанная на Rust, разработанная для приложений САПР/САИ/САПМ.

## Что такое BrepRs?

BrepRs — это библиотека граничного представления (BRep), которая предоставляет:

- **Высокопроизводительные** операции с геометрией и топологией
- **Поддержку нескольких языков** через языковые привязки
- **Полный API** для 3D-моделирования
- **Кросс-платформенную** совместимость
- **Поддержку WebAssembly** для браузерных приложений

## Основные характеристики

### Основные функции

- **Булевые операции** (объединение, вычитание, пересечение, секция)
- **Поверхностное моделирование** (плоскости, цилиндры, сферы, конусы, тори)
- **Топологические операции** (манипуляция вершинами, ребрами, гранями, оболочками, телами)
- **Генерация сетки** и манипуляция
- **Ввод/вывод файлов** (STEP, IGES, STL, OBJ, glTF)
- **Поддержка интернационализации**

### Языковые привязки

BrepRs предоставляет привязки для нескольких языков программирования:

- **Python** - с использованием PyO3
- **C/C++** - с использованием FFI
- **Fortran** - с использованием C FFI
- **Java** - с использованием JNI
- **Node.js** - с использованием NAPI-RS
- **PHP** - с использованием ext-php-rs

### Производительность

- **Основано на Rust** для максимальной производительности
- **Параллельная обработка** с Rayon
- **Безопасный в памяти** дизайн
- **Операции без копирования** (где возможно)
- **Оптимизированные алгоритмы** для часто используемых операций

## Варианты использования

BrepRs подходит для широкого спектра приложений:

- **Системы САПР** - компьютерное проектирование
- **Системы САПМ** - компьютерное управление производством
- **Системы САИ** - компьютерное моделирование
- **3D-визуализация** - рендеринг и симуляция
- **Обработка геометрии** - анализ и манипуляция
- **Аддитивное производство** - подготовка к 3D-печати
- **Робототехника** - планирование пути и обнаружение столкновений
